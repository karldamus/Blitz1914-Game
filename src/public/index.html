Blitz 1914

Blitz 1914 is a multiplayer game that takes place during World War I. The game is a real time strategy game where players will join simultaneously and be able to achieve world domination within 30 minutes to an hour. The game features a variety of units, including infantry, artillery, tanks, battleships, and planes. The game is designed to be fast-paced and strategic. Blitz 1914 is free to play and can be played in your web browser
